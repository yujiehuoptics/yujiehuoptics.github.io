PoC: Data Streaming for a Multi-channel Embedded Data Acquisition System in Field Industrial Applications



Introduction
The client requires unmanned and unattended underwater multi-channel, long-term data acquisition. To ensure the smooth storage of data and actual power consumption calculations in the computing system, a validation setup is needed.

System Overview
Hardware: cRIO-9082 cRIO-9159 NI-9234 PXIe-4430
Software: LabVIEW 2015, Real-Time 2015, FPGA 2015

Solution
1. Verify the capability of cRIO-9082 and cRIO-9159 to perform data acquisition using 13 NI-9234 modules at a rate of 51.2 kS/s per channel, totaling 52 channels.
2. Validate the ability of cRIO-9082 and cRIO-9159 to stream data from 52 channels, amounting to 10.6 MB/s, to an external portable hard drive.
3. Measure the power consumption of the system during streaming disk operations using 2TB and 1TB portable hard drives.

Result
1. Data acquistion
Since 13 units of NI-9234 modules couldn't be located, an alternative approach was adopted, involving the repeated transfer of identical data 13 times within the FPGA to simulate the actual data acquisition speed. Validation conducted at the host end yielded a transmission speed of 10.6 MB/s. Whether utilizing the 9234 modules for acquisition on the 9082 or the 9159, the corresponding acquisition speed was achieved, effectively capturing data waveforms.


2. Streaming
Conducting streaming disk tests with 1TB and 2TB portable hard drives, employing both TDMS and binary file formats for the streaming process. Achieving a streaming speed of up to 10.6 MB/s per second, and each file is saved with a size of 1GB.

3. Power Consumption

For the 9082, the power consumption of the 1TB portable hard drive is higher than that of the 2TB drive. This difference could be attributed to the 2TB drive being relatively newer and having a more efficient power management chipset.
